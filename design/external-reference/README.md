# External Reference Notes

This folder documents external inspiration for the current Keystone UI direction.

## Current External Reference Set

The committed external reference set in this repo is:

- `screen-1/`
- `screen-2/`
- `screen-3/`
- `screen-4/`

These are the source screenshots and companion HTML captures that were being referenced during the Keystone UI direction work.

This reference set is useful for:

- shell composition
- left-heavy navigation with a dominant center workspace
- chat-first interaction patterns
- pairing a working surface with a right-side inspector

That reference is not a literal spec for Keystone.

We are intentionally adapting it to Keystone's workflow:

- project and run navigation on the far left
- artifact tree on the inner left
- central mode switch between `Agent Chat` and `Workflow DAG`
- DAG node selection that moves the user into task-scoped chat
- release and evidence artifacts in the inspector

## Relationship To The Keystone Target

These screens are not the Keystone target and should not be treated as a literal product spec.

The adapted Keystone direction lives under `../target-reference/`, and the design rules that govern that translation live in `../design-guidelines.md`.
